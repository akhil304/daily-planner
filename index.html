Connect Cloud Sync

Sync across all your devices in real time with a free Supabase database.

Step 1 — Create free Supabase project

Go to supabase.com, sign up free, create a project.

Step 2 — Run this SQL once

In SQL Editor paste and run:create table if not exists planner(key text primary key, value text, updated_at timestamptz default now()); alter table planner enable row level security; create policy "public rw" on planner for all using (true) with check (true);

Step 3 — Paste credentials

Project Settings → API → copy Project URL and anon key.

Your data stays in your own Supabase project.
