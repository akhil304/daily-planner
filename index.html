Sync across all your devices with a free Supabase database.

Step 1 — Create free project

Go to supabase.com, sign up, create a project.

Step 2 — Run SQL once

In SQL Editor run:
create table if not exists planner(key text primary key, value text, updated_at timestamptz default now()); alter table planner enable row level security; create policy "public rw" on planner for all using (true) with check (true);

Step 3 — Paste credentials

Project Settings → API → copy URL and anon key.

Data stays in your own Supabase project.
