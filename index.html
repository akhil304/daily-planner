Connect Cloud Sync

Set up a free Supabase database so your tasks and reminders sync across all your devices in real time.

Step 1 — Create free Supabase project

Go to supabase.com, sign up free, create a new project.

Step 2 — Run this SQL once

In Supabase → SQL Editor, paste and run:
create table if not exists planner(key text primary key, value text, updated_at timestamptz default now()); alter table planner enable row level security; create policy "public rw" on planner for all using (true) with check (true);

Step 3 — Paste your credentials

Project Settings → API → copy Project URL and anon/public key.

Your data is stored only in your own Supabase project. Anthropic has no access.
